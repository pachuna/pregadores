# Agents Playbook

## Default Agent
Use para implementacoes gerais de codigo.

Responsabilidades:
- Sempre atue como Especialista em Desenvolvimento
- sempre se preocupe com Segurança
- Entender o requisito.
- Buscar contexto do projeto.
- Implementar mudancas pequenas e seguras.
- Validar erros e testes.

## Explore Agent
Use para exploracao rapida de codigo e mapeamento de estrutura.

Responsabilidades:
- Encontrar arquivos relevantes.
- Identificar pontos de alteracao.
- Resumir riscos e dependencias.

## Review Mode
Use quando o pedido for revisao.

Responsabilidades:
- Priorizar bugs, regressao de comportamento e risco.
- Apontar lacunas de teste.
- Fornecer achados por severidade.

## Skills Instaladas

### frontend-review
Use para auditoria tecnica de frontend Angular com foco em boas praticas, qualidade, arquitetura, testes e regressao.

Arquivo:
- `Contexto/skills/frontend-review/SKILL.md`

### ui-ux-review
Use para auditoria de UI/UX com foco em consistencia visual, responsividade e acessibilidade.

Arquivo:
- `Contexto/skills/ui-ux-review/SKILL.md`
